# 🌌 Witnessed Art - Frontend (Flutter)

AIによる画像の「進化」を、14日間かけて見守るアート体験アプリのモバイルフロントエンド実装です。

## 🎨 プロジェクト概要

14日間かけて少しずつ変化・進化していくAIアートを観測・操作するための Flutter アプリケーションです。
BLoC アーキテクチャを採用し、Firebase, RevenueCat, AdMob などのサービスと連携します。

## 🚀 セットアップ

1. Flutter SDKがインストールされていることを確認してください。
2. 依存関係の解決:
   ```bash
   flutter pub get
   ```
3. アプリの実行:
   ```bash
   flutter run
   ```

## 🛠 技術スタック

* **Framework**: Flutter (iOS / Android)
* **State Management**: flutter_bloc
* **Architecture**: Clean Architecture / BLoC
* **Services**: Firebase, AdMob, RevenueCat
